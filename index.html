Two things: Learn + Game

Code for a week. Stack for prizes.

Learn/Game is a browser-only coding bootcamp and arcade practice room. The Learn side gives long, guided lessons on types, variables, logic, functions, arrays, objects, loops, DOM work, debugging, and mini projects. The Game side is a Stacker-style practice game: moving blocks slide side to side, you stop them, and only the overlapping blocks survive as you climb toward minor and major prize zones.

Rules copied into this practice version

Start with three blocks. If you stop partly off the stack, the row shrinks. If there is no overlap, the screen turns red and shows retry. Choose Minor Run to cash out at the minor line with a green retry screen, or Major Run to pass the minor line and climb to the yellow jackpot retry screen.
