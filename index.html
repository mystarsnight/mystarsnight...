Two things: Learn + Game

Code for a week. Stack for prizes.

Learn/Game is a browser-only coding bootcamp and arcade practice room. The Learn side gives long, guided lessons on types, variables, logic, functions, arrays, objects, loops, DOM work, debugging, and mini projects with five tasks per day for seven days. The Game side is a Stacker-style practice game: moving blocks slide side to side, you stop them, and only the overlapping blocks survive as you climb toward minor and major prize zones.

Rules copied into this practice version

Start with three blocks. The row slides side to side, bounces off the edges, and speeds up each level. When you stop, only the blocks overlapping the row below stay lit. No overlap turns the screen red. The minor line asks Prize? like the real cabinet: take green minor, or keep playing for yellow major.
